# 个人博客前端界面 PRD（科技增强版）

## 一、项目概述

### 1.1 项目背景
创建一个融合东方禅意与数字科技的个人博客，不仅是内容展示窗口，更是一件可交互的数字艺术品。通过粒子、光影与微交互，让每一次阅读成为穿梭于静谧与未来之间的旅程。

### 1.2 设计方向
- **Tone**: 「诗意科技 · 静谧未来主义」—— 以留白为画布，用动态粒子、柔和光影与玻璃质感构筑层次，在极简中呈现科技的温度。
- **Differentiation**: 首创"数字水墨"阅读体验：背景粒子随滚动如墨痕晕染，卡片悬浮带有景深光效，文章进度以光弧示意，让科技隐于诗意之中。
- **关键词**: 静谧、留白、粒子、光影、玻璃态、动态韵律、AI 辅助

### 1.3 目标用户
- 内容创作者/独立博主
- 追求美学体验的阅读爱好者
- 科技、设计行业同行及潜在合作伙伴

## 三、设计规范（科技增强版）

### 3.1 色彩系统
| 色彩类型 | 浅色主题 | 深色主题（默认） | 用途 |
|----------|---------|---------------|------|
| 主色 | #1A1A2E | #E0E7FF | 文字、标题 |
| 辅色·霓虹蓝 | #4FACFE | #00F0FF | 链接、高亮、粒子主色 |
| 辅色·霓虹紫 | #9D4EDD | #BD00FF | 强调、光晕、按钮渐变 |
| 背景 | #F5F3F0 米白 | #0B0C10 深邃黑 | 全局背景 |
| 卡片/玻璃态 | rgba(255,255,255,0.6) + 模糊 | rgba(20,20,35,0.7) + 模糊 | 毛玻璃卡片 |
| 光效渐变 | linear-gradient(135deg, #4FACFE, #9D4EDD) | 同左，高亮使用 | 按钮、进度光环 |

### 3.2 字体系统
新增 **科技数字字体** 用于数据、时间等：
- **数字/代码**: `Space Mono` / `JetBrains Mono`
- **标题**: `Noto Serif SC` (保留人文诗意)
- **正文**: `Inter` + `Noto Sans SC` (保证清晰度)
- **特效文字**: 可选用 `Orbitron` 用于极少数未来感标识

### 3.3 间距与圆角
- 全局圆角 12px，卡片采用 16px 圆角，按钮及输入框采用完全圆形或大圆角，强化友好科技感。
- 间距增加**液态间距**概念：部分元素间使用负 margin 或重叠，形成层次错落。

### 3.4 阴影与发光
- 卡片阴影：`0 8px 32px rgba(0,240,255,0.1)`，hover 时加深并增加同色系辉光。
- 文字发光：标题在深色模式下 `text-shadow: 0 0 10px rgba(0,240,255,0.5)`。

## 七、性能与体验优化

- **首屏加载优化**：粒子背景懒加载，优先保证内容渲染；Three.js 场景使用轻量模型。
- **自适应粒子质量**：根据设备性能动态调整粒子数量和特效强度。
- **图片全部懒加载**，并支持 WebP 格式，结合模糊占位。
- **代码分割**：粒子系统、3D 组件、AI 模块均为独立 chunk。
- **内存管理**：Canvas/Three.js 在页面隐藏时暂停渲染循环。
