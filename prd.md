# 个人博客前端界面 PRD

## 一、项目概述

### 1.1 项目背景
创建一个具有独特设计感的个人博客前端界面，用于展示博主的文章、作品和个人信息。

### 1.2 设计方向
- **Tone**: 「诗意极简主义」—— 融合东方美学与现代设计，以留白、韵律感和层次感为核心
- **Differentiation**: 独特的垂直阅读体验，配合沉浸式的视觉叙事，让读者在阅读中获得艺术享受
- **关键词**: 静谧、留白、韵律、层次、诗意

### 1.3 目标用户
- 内容创作者/博主
- 阅读爱好者
- 行业同行/潜在合作伙伴

| 功能点 | 描述 | 优先级 |

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 文章标题 | 大字号、醒目展示 | 高 |
| 文章元信息 | 作者、日期、阅读时间、标签 | 高 |
| 文章内容 | Markdown渲染，支持代码高亮 | 高 |

### 2.4 分类/标签页

### 2.5 关于页
| 功能点 | 描述 | 优先级 |

## 三、设计规范

### 3.1 色彩系统
| 色彩类型 | 色值 | 用途 |
|----------|------|------|
| 主色 | #2C3E50 | 文字、标题、主要交互元素 |
| 辅色 | #3498DB | 链接、按钮、高亮元素 |
| 强调色 | #E74C3C | 重要提示、CTA |
| 背景色 | #FAFAFA | 页面背景 |
| 卡片色 | #FFFFFF | 内容卡片 |
| 分割线 | #EEEEEE | 分隔元素 |
| 深色模式主色 | #ECF0F1 | 深色模式文字 |
| 深色模式背景 | #1A1A2E | 深色模式背景 |

### 3.2 字体系统
| 字体类型 | 字体名称 | 用途 | 字重 |
|----------|----------|------|------|
| 标题字体 | Noto Serif SC | 文章标题、Hero标题 | 400/700 |
| 正文字体 | Noto Sans SC | 正文内容、导航 | 300/400/500 |
| 等宽字体 | JetBrains Mono | 代码块 | 400 |

### 3.3 间距系统
| 间距类型 | 值 | 用途 |
|----------|-----|------|
| xs | 4px | 微小间距 |
| sm | 8px | 小间距 |
| md | 16px | 中等间距 |
| lg | 24px | 大间距 |
| xl | 32px | 特大间距 |
| xxl | 48px | 超大间距 |

### 3.4 布局规范
- **最大宽度**: 1200px (内容区域)
- **侧边栏宽度**: 300px
- **响应式断点**: 1024px (桌面端), 768px (平板), 480px (手机)

## 四、页面结构

### 4.1 首页 (`/`)
```
├── Header (固定导航)
│   ├── Logo
│   ├── 导航菜单
│   └── 主题切换
├── Hero Section
│   ├── 博主头像
│   ├── 博主名称
│   ├── 个人简介
│   └── CTA按钮
├── Featured Articles
│   └── 精选文章卡片
├── Article List
│   ├── 筛选器
│   └── 文章卡片网格
├── Footer
│   ├── 版权信息
│   ├── 社交媒体链接
│   └── 导航链接
```

### 4.2 文章详情页 (`/article/:id`)
```
├── Header
├── Article Header
│   ├── 分类标签
│   ├── 标题
│   ├── 元信息
│   └── 封面图
├── Article Content
│   ├── 目录导航(侧边)
│   └── 正文内容
├── Article Footer
│   ├── 上一篇/下一篇
│   └── 标签
├── Comments Section
└── Footer
```

### 4.3 关于页 (`/about`)
```
├── Header
├── About Hero
│   ├── 头像
│   ├── 姓名
│   └── 头衔
├── Biography
│   └── 个人简介
├── Skills
│   └── 技能标签/进度条
├── Timeline
│   └── 履历时间线
├── Contact
│   └── 联系方式
└── Footer
```

## 五、交互设计

### 5.1 动画效果
| 场景 | 效果 | 触发方式 |
|------|------|----------|
| 页面加载 | 元素淡入 + 向上滑动 | 页面进入 |
| 导航栏滚动 | 背景渐变、阴影出现 | 向下滚动 |
| 文章卡片悬停 | 浮起效果、阴影加深 | 鼠标悬停 |
| 按钮悬停 | 颜色过渡、缩放 | 鼠标悬停 |
| 页面切换 | 淡入淡出过渡 | 路由切换 |
| 阅读进度 | 顶部进度条平滑增长 | 滚动 |

### 5.2 微交互
- 标签点击筛选文章
- 搜索框实时反馈
- 返回顶部按钮渐显
- 评论输入实时字数统计
